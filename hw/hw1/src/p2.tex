\section*{Problem 2}

We are running programs where values of type int are 32 bits. They are represented in two's complement, and they are right shifted arithmetically. Values of type unsigned are also 32 bits. We generate arbitrary values x and y, and convert them to unsigned values as follows:

\begin{lstlisting}[frame=tb, language=C++]
/* Create some arbitrary values */
int x = random();
int y = random();
/* Convert to unsigned */
unsigned ux = (unsigned) x;
unsigned uy = (unsigned) y;
\end{lstlisting}

\noindent For each of the following C expressions, either (1) argue that it is true (evaluates to True) for all values of x and y, or (2) give values of x and y for which it is false (evaluates to False):

\begin{enumerate}
    \item \code{(x < 0) == (-x > 0)}
    \item \code{{\raise.17ex\hbox{$\scriptstyle\mathtt{\sim}$}}x + {\raise.17ex\hbox{$\scriptstyle\mathtt{\sim}$}}y < {\raise.17ex\hbox{$\scriptstyle\mathtt{\sim}$}}(x + y)}
    \item \code{((x >> 6) << 6) <= x}
    \item \code{(ux + uy) == -(unsigned)(-y - x)}
\end{enumerate}
